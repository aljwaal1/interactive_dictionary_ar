# قاموس تفاعلي

تطبيق Flutter Android خفيف مبني على قالب APK المباشر.

## المزايا

- واجهة عربية RTL.
- تصميم مستقل يناسب فكرة التطبيق.
- يعمل محليًا في النسخة الأولى بدون API خارجي.
- زر نسخ النتيجة.
- صفحة مراسلة المطور بزر نسخ الرسالة فقط.

## البناء

يتم البناء عبر GitHub Actions فقط من الملف:

`.github/workflows/build-apk.yml`

## ملف APK النهائي

`apk/interactive-dictionary.apk`
